# Jazz Full

**Status:** sketch

ii–V–I in C major, 4 loops, with a walking bass and swing drums. The walking bass
approaches each root chromatically on the last beat of each bar (Dm7 → G7 via Ab2,
G7 → Cmaj7 via Db3, Cmaj7 → Dm7 via Eb2). Three tracks.
