# Jazz Progression

**Status:** sketch / building block

ii–V–I in C major at 120 BPM. Three block chords (Dm7 → G7 → Cmaj7), one track.
The simplest possible piece — used as the harmonic foundation that `jazz_full` and
`jazz_with_drums` build on top of.

## Possible next moves
